# Magic Set Editor - Shared Cloud

Welcome to the shared Magic Set Editor (MSE2) cloud environment! 

This repository contains a fully self-contained, portable version of MSE2 that automatically syncs custom sets with other creators via GitHub, tracks set goals, and injects creator metadata into cards.

## How to Install and Run

You do NOT need to install Git, Node, or any other software. Everything is included in this package.

1. **Download the Package:**
   - Go to this link: `https://github.com/basscosauce-beep/MSE2-Shared-Editor`
   - Click the green **`<> Code`** button in the top right.
   - Click **`Download ZIP`**.

2. **Extract:**
   - Extract the downloaded ZIP file to a permanent location on your computer (e.g., your Desktop or Documents folder). Do not run it directly from inside the ZIP file!

3. **Launch:**
   - Open the extracted folder and double-click **`Mtg Card edditer.bat`**.
   - The very first time you run it, a prompt will ask you for your name or initials. This is used to automatically fill the "By:" column when you create new cards.

4. **Start Creating:**
   - MSE2 will open automatically with your team's shared sets.
   - The launcher will silently pull any updates your team has made from the cloud before opening.
   - You can access the Account Settings or the Set Goal Tracker by clicking the new buttons in the top menu bar of MSE2!

> **Note:** If you get a Windows SmartScreen warning when running the launcher, click "More Info" and then "Run Anyway". It only appears because this is a custom-made script.
